### 🚀 Featured Data Science Projects

#### 📈 1. Walmart Sales Forecasting & Demand Trends
* **Objective:** Developed predictive modeling pipelines to capture seasonal demand fluctuations and COVID-19 anomalies.
* **Stack:** `Time Series Analysis`, `ARIMA`, `SARIMA`, `Matplotlib`, `Seaborn`
* **Impact:** Delivered structured visual trend analyses mapping out historical and future demand cycles.

#### 🧠 2. Real-Time AI Face Detection System
* **Objective:** Built an edge computer vision framework localizing human faces in active video feeds.
* **Stack:** `Python`, `OpenCV`, `Computer Vision`, `Image Processing`

#### 🛍️ 3. Customer Churn Prediction Engine
* **Objective:** Designed an end-to-end classification strategy determining customer churn hazards using behavioral telemetry.
* **Stack:** `Logistic Regression`, `Random Forest`, `Feature Engineering`, `EDA`
* **Impact:** Pinpointed crucial retention vectors directly affecting customer life-cycle management.

#### 💬 4. Movie Review Sentiment Analyzer (IMDb)
* **Objective:** Constructed a text processing pipeline to analyze sentiment vectors across 50,000 text data points.
* **Stack:** `TF-IDF`, `DistilBERT`, `VADER`, `SVM`, `Streamlit`
* **Impact:** Attained ~89% baseline accuracy and deployed via an interactive real-time prediction application.

#### 🎬 5. Collaborative Filtering Recommendation Engine
* **Objective:** Built a matrix-factorization movie recommendation infrastructure mirroring major streaming ecosystems.
* **Stack:** `Singular Value Decomposition (SVD)`, `Collaborative Filtering`, `Matrix Factorization`
